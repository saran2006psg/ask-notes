WHAT WE ARE DEVELOPING:
Developed a Retrieval-Augmented Generation (RAG) based College Notes Assistant that enables students to query multiple academic notes across subjects such as DBMS, Operating Systems, and Computer Networks.

Implemented PDF ingestion, document chunking, semantic embeddings, FAISS-based vector retrieval, and LLM-powered answer generation with source-aware responses. Enhanced retrieval quality using metadata filtering and multi-document semantic search, enabling efficient knowledge discovery across large academic document collections.
